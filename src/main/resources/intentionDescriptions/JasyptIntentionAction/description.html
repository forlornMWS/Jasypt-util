Encrypts selected text using Jasypt encryption library.

This intention allows you to quickly encrypt sensitive information in your code or configuration files using the Jasypt encryption library, which is commonly used in Java applications for securing configuration values.

Select the text you want to encrypt, invoke the intention, and provide a secret key and algorithm to encrypt the value: 


// Before:
spring.datasource.password=my secret password

// After:
spring.datasource.password=ENC(G6N048DiLbTjUBOMrNhGAvxCyPwYQV1sVoQtQUiZYmytP34mbMy5DA==)

This intention is particularly useful when working with Spring applications that use Jasypt for property encryption.

The encrypted value can be configured with or without the "ENC()" wrapper, depending on your application's requirements.

Support PBE Algorithm

  • PBEWithMD5AndDES
  • PBEWithSHA1AndDES
  • PBEWithSHA1AndRC2_40
  • PBEWithSHA1AndRC4_128
  • PBEWithSHA1AndRC4_40
  • PBEWithHmacSHA512AndAES_256
  • PBEWithHmacSHA512AndDES3
  • PBEWithHmacSHA512AndRC2_128
  • PBEWithHmacSHA512AndRC4_128
  • PBEWithHmacSHA512AndRC4_40

To customize the default encryption settings for this plugin, visit Plugin Settings.
